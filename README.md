# 핵노답 아카이브(Hacknodap Archive)

소규모 그룹의 활동 기록을 관리하고 보기 편하게 활동 기록 타임라인을 지원하는 웹앱.

이런 게 있으면 핵노답이라도(본인 포함) 뭐라도 쓰지 않을까?
